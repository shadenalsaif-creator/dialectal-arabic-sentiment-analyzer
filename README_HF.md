# Dialectal Arabic Sentiment Analyzer

See the full project README below. This Space runs the Streamlit app directly.

> **Note on the model:** by default the Space runs the public CAMeLBERT-DA
> base model so it works with no extra setup. To serve the fine-tuned model,
> push the `models/camelbert-finetuned/` folder to this Space using Git LFS,
> or load it from the Hugging Face Hub.
